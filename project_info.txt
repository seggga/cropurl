2) Пакеты в составе приложения:

    - restapi - сервер API
        - инициализация
        - остановка

    - handler - фукнции обработчика вэб-запросов
        - создание новой пары short ID -> url
        - удаление пары short ID -> url
        - редирект по short ID
        - просмотр статистики использования short ID

    - storage - задает интерфейс для взаимодействия с выбранным хранилищем (файл / БД / мапа..)
        - запись данных
        - чтение данных 
            - чтение статистики обращений
            - чтение длинного url по известному короткому ID
        - проверка дубликатов (короткий url уже занят)
