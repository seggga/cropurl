2) Пакеты в составе приложения:
    - endpoint - отвечает за сетевое взаимодействие с пользователями
        - принимает запросы
        - отправляет ответы

    - storage - задает интерфейс для взаимодействия с выбранным хранилищем (файл / БД / мапа..)
        - формат данных:
            - короткий url
            - длинный url
            - статистика обращений
            - описание (комментарий пользователя)

        - реализуемые функции
            - запись данных
                - кол-во переходов +1
                - создание новой пары длинный-короткий
            
            - чтение данных 
                - чтение статистики обращений
                - чтение длинного url по известному короткому ID
            
            - проверка дубликатов (короткий url уже занят)
